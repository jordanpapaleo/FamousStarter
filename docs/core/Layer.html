Layer

Layer [constructor] 

Layers manage a set of components or renderables. Components are expected to expose a kill method. Optionally they can expose a clean method which will be called as soon as the layer they are registered on is being cleaned and they are being dirty.

new Layer()

Instance methods

.clear()

Clears all components by killing all components having kill set to a truthy value and removing them from the Layer.

.requestId()

Returns an id which can be used in order to register a new component using registerAt.

.registerAt(id, component)

Registers the passed in component on the specified id. Does not dirty the component.
