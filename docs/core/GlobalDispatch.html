GlobalDispatch

GlobalDispatch [constructor] 

GlobalDispatch is being used in order to manage scene graph events. It routes and manages events being registered on specific nodes, but also provides the possibility to globally register event listeners on the whole scene graph.

new GlobalDispatch()

Instance methods

.targetedTrigger(path, key, ev)

Triggers the event defined by key (type of event) on a Node in the scene graph hierarchy defined as a series of RenderProxy id's composing the path exactly that Node.

Parameters:

NameTypeDescription
pathString

path to the respective Node in the scene graph defined as series of RenderProxy ids.

keyString

arbitrary event type (e.g. "click", "move")

evObject

event object

.targetedOn(path, key, cb)

Registers an event listener for an event being emitted on a specific node in the scene graph hierarchy.

Parameters:

NameTypeDescription
pathString

path to the respective Node in the scene graph defined as series of RenderProxy ids.

keyString

arbitrary event type (e.g. "click", "move")

cbfunction

callback function to be invoked whenever the event defined by path and key is being triggered.

.targetedOff(path, key, cb)

Removes a previously via targetedOn registered event listener,

Parameters:

NameTypeDescription
pathString

path to the respective Node in the scene graph defined as series of RenderProxy ids.

keyString

arbitrary event type (e.g. "click", "move")

cbfunction

previously via targetedOn registered callback function

.globalOn(path, key, cb)

Globally registers an event listener. Listeners registerd using this method can not be triggered by their path, but only globally by the event they have been registered on.

Parameters:

NameTypeDescription
pathString

path to the respective Node in the scene graph defined as series of RenderProxy ids. This will only be used in order to extract depth of this Node in the scene graph.

keyString

arbitrary event type (e.g. "click", "move")

cbfunction

callback function to be called whenever an event in the SceneGraph

.globalOn(path, key, cb)

Removed an event listener that has previously been registered using globalOn.

Parameters:

NameTypeDescription
pathString

path to the respective node the listener has been registered on. Used inly to extract the depth from it.

keyString

event type used for registering the event listener

cbfunction

registered callback function

Triggers all global event listeners registered on the specified type.

Parameters:
