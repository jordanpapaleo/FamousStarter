Opacity

Opacity [constructor] [private]

Initilizes the Opacity primitive by settings its value to 1 (default value). Hierarchically setting opacity does not affect the final, local opacity being returned. Rather, this functionality needs to be implemented in the corresponding render targets (e.g. DOM has blending by default).

new Opacity()

Instance methods

.set(value)

Sets, activates and dirties the internal notion of opacity being read by the RenderContext.

Parameters:

NameTypeDescription
valueOpacity

new opacity to be set

.setDirty()

Dirties the opacity. This forces the RenderContext to trigger the opacity event on the next invocation of the update method on RenderContext.

.clean()

Cleans the opacity. This sets its dirty flag to false, thus no longer reading it in update of the RenderContext.
