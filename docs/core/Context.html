Context

Context [constructor] [private]

Context is the top-level node in the scene graph (= tree node). As such, it populates the internal MessageQueue with commands received by subsequent child-nodes. The Context is being updated by the Clock on every FRAME and therefore recursively updates the scene grpah.

new Context(selector)

Constructor parameters:

NameTypeDescription
selectorString

query selector used as container for context

Instance methods

.addChild()

Adds a child to the internal list of child-nodes.

.removeChild(node)

Removes a node returned by addChild from the Context's immediate children.

Parameters:

NameTypeDescription
nodeNode

node to be removed

.update()

Recursively updates all children.

.getRenderPath()

Returns the selector the Context is attached to. Terminates recursive getRenderPath scheduled by RenderProxy.

.receive(command)

Appends the passed in command to the internal MessageQueue, thus scheduling it to be sent to the Main Thread on the next FRAME.

Parameters:

NameTypeDescription
commandObject

command to be enqueued

._receiveContextSize(size)

Method being executed whenever the context size changes.

Parameters:

NameTypeDescription
sizeArray

new context size in the format [width, height]
