MessageQueue

MessageQueue [constructor] 

Used for scheduling messages to be sent on the next FRAME. The MessageQueue is being cleared after each postMessage in the Famous singleton.

new MessageQueue()

Instance methods

.enqueue(message)

Pushes a message to the end of the queue to be sent on the next FRAME.

Parameters:

NameTypeDescription
messageObject

message to be appended to the queue

.getAll()

Returns an array of all messages currently scheduled for the next FRAME.

.clear()

Empties the queue.
