Transform

Transform [constructor] 

Transform is an object that is part of every RenderContext, Align and its derivatives Origin and MountPoint. It is responsible for updating its own notion of position in space and incorporating its parent information.

new Transform()

Instance methods

.getGlobalMatrix()

Return the transform matrix that represents this Transform's values being applied to it's parent's global transform.

.getLocalVectors()

Return the vectorized information for this Transform's local transform.
