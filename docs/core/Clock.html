core/Clock.js

Clock

Clock [constructor] 

Equivalent of an Engine in the Worker Thread. Used to synchronize and manage time across different Threads.

new Clock()

Instance methods

.step(time)

Updates the internal clock time.

Parameters:

NameTypeDescription
timeNumber

high resolution timstamp used for invoking the update method on all registered objects

.update(updateable)

Registers an object to be updated on every frame.

Parameters:

NameTypeDescription
updateableObject

Object having an update method

updateable.updatefunction

update method to be called on every step

.noLongerUpdate(updateable)

Deregisters a previously using update registered object to be no longer updated on every frame.

Parameters:

NameTypeDescription
updateableObject

Object previously registered using the update method

.getTime(time)

Returns the internal clock time.

Parameters:

NameTypeDescription
timeNumber

high resolution timstamp used for invoking the update method on all registered objects

.nextStep(updateable)

Registers object to be updated once on the next step. Registered updateables are not guaranteed to be unique, therefore multiple updates per step per object are possible.

Parameters:

NameTypeDescription
updateableObject

Object having an update method

updateable.updatefunction

update method to be called on the next step

.setTimeout(callback, delay)

Wraps a function to be invoked after a certain amount of time. After a set duration has passed, it executes the function and removes it as a listener to 'prerender'.

Parameters:

NameTypeDescription
callbackfunction

function to be run after a specified duration

delayNumber

milliseconds from now to execute the function

.clearTimer(callback)

Removes previously via Clock#setTimeout or Clock#setInterval registered callback function

Parameters:

NameTypeDescription
callbackfunction

previously via Clock#setTimeout or Clock#setInterval registered callback function

.setInterval(callback, duration)

Wraps a function to be invoked after a certain amount of time. After a set duration has passed, it executes the function and resets the execution time.

Parameters:

NameTypeDescription
callbackfunction

function to be run after a specified duration

durationNumber

interval to execute function in milliseconds
