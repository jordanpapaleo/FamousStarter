.registerCurve(name, curve)

Registers a given curve to be available in subsequent transitions by adding it to the interal dictionary of registered curves.

Parameters:

NameTypeDescription
nameString

unique name for later access

curvefunction

function of one numeric variable mapping [0,1] to range inside [0,1]

.unregisterCurve(name)

Unregisters the curve registered under the given name by removing it from the internal dictionary of registered curves. This won't effect currently active transitions.

Parameters:

NameTypeDescription
nameString

name of curve

.getCurve(name)

Returns the easing curve with the given name.

Parameters:

NameTypeDescription
nameString

name of curve

.getCurves()

Retrieves the names of all previously registered easing curves.
