TweenTransition

TweenTransition [constructor] 

A state maintainer for a smooth transition between numerically-specified states. Example numeric states include floats or Transfornm objects.

An initial state is set with the constructor or set(startValue). A corresponding end state and transition are set with set(endValue, transition). Subsequent calls to set(endValue, transition) begin at the last state. Calls to get(timestamp) provide the _interpolated state along the way.

Note that there is no event loop here - calls to get() are the only way to find out state projected to the current (or provided) time and are the only way to trigger callbacks. Usually this kind of object would be part of the render() path of a visible component.
