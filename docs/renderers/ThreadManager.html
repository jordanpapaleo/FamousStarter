ThreadManager

ThreadManager [constructor] 

The ThreadManager is being updated by an Engine by consecutively calling its update method. It can either manage a real Web-Worker or the global Famous core singleton.

new ThreadManager(thread, compositor)

Constructor parameters:

NameTypeDescription
threadFamous

The thread being used to receive messages from and post messages to. Expected to expose a WebWorker-like API, which means providing a way to listen for updates by setting its onmessage property and sending updates using postMessage.

compositorCompositor

an instance of Compositor used to extract enqueued draw commands from to be sent to the thread

Instance methods

.update(time)

Update method being invoked by the Engine on every requestAnimationFrame. Used for updating the notion of time within the managed thread by sending a FRAME command and sending messages to

Parameters:

NameTypeDescription
timeNumber

unix timestamp to be passed down to the worker as a FRAME command
