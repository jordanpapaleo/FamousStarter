WebGLRenderer

WebGLRenderer [constructor] [private]

WebGLRenderer is a private class that manages all interactions with the WebGL API. Each frame it receives commands from the compositor and updates its registries accordingly. Subsequently, the draw function is called and the WebGLRenderer issues draw calls for all meshes in its registry.

new WebGLRenderer(canvas)
