Texture

Texture [constructor] 

Texture is a private class that stores image data to be accessed from a shader or used as a render target.

new Texture()

Instance methods

.bind(unit)

Binds this texture as the selected target.

Parameters:

NameTypeDescription
unitNumber

The texture slot in which to upload the data.

.unbind(unit)

Erases the texture data in the given texture slot.

Parameters:

NameTypeDescription
unitNumber

The texture slot in which to clean the data.

.setImage(img)

Replaces the image data in the texture with the given image.

Parameters:

NameTypeDescription
imgImage

The image object to upload pixel data from.

.setArray(input)

Replaces the image data in the texture with an array of arbitrary data.

Parameters:

NameTypeDescription
inputArray

Array to be set as data to texture.

.readBack()

Dumps the rgb-pixel contents of a texture into an array for debugging purposes

Parameters:

x-offsetNumber

between texture coordinates and snapshot

y-offsetNumber

between texture coordinates and snapshot

x-depthNumber

of the snapshot
