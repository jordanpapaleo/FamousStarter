History

History [constructor] 

A stateless shim for hash routing. Used by router. Supports hash bang routing and HTML5 pushState. Falls back to hash bang urls when pushState is not supported. Implements subset of W3C spec in respect to http://www.w3.org/TR/2011/WD-html5-20110113/history.html#history-0
