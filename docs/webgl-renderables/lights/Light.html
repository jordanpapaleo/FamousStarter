.setColor(r, g, b, options)

Changes the color of the light, using 'Color' as its helper. It accepts an optional options parameter for tweening colors. Its default parameters are in RGB, however, you can also specify different inputs. setColor(r, g, b, option) setColor('rgb', 0, 0, 0, option) setColor('hsl', 0, 0, 0, option) setColor('hsv', 0, 0, 0, option) setColor('hex', '#000000', option) setColor('#000000', option) setColor('black', option) setColor(Color)
