Famous Platform API Documentation

Famous is a platform for building user-interfaces. It's written in JavaScript and features high-performance rendering capabilities across both DOM and WebGL. Included is a fully integrated 3D physics engine and a handful of other utilities for UI creation.

Famous was built from the ground up to bring 60 FPS animations to any screen. Because it's rooted in web technilogies, Famous applications can go anywhere: browser, device, jumbotron. Famous' internal architecture takes inspiration from AAA game engines, breaking out of document-based UI limitations with a full fledged scene graph that developers can access at a low level.


Understanding the scene graph is key to understanding the Famous platform. In Famous, all display elements are associated with a render node which can be decorated with visual characteristics such as size, rotation, position, and style. Render nodes are maintained by Famous as vertices in a tree data structure. Its hierarchy is used to group visual elements and apply additive transformations.


Although the Famous platform is built in web technologies, building UIs with Famous is very much unlike traditional web development. It might seem unusual, especially if you're accustomed to building UIs with conventional HTML, CSS, and JavaScript. Notions like normal flow and floated elements, the bread and butter of modern websites, are peripheral or avoided completely when using Famous.


Getting started

To start developing with Famous, we recommend beginning with one of the seed projects which you can clone to your local workstation and then modify. Begin by using Git to clone down Famous' main seed repository:

$ git clone git@github.com:Famous/seed.git

Once the repository has finished being cloned down, change directories into the seed project, and then run the following command to install dependencies (you'll need to make sure you have Node.js installed before attempting this command):

$ npm install

Finally, start up the local development server by running the following command. Once the development server begins running, you should be able to visit http://localhost:1337 in your favorite browser and see a spinning Famous logo. If not, please visit our section on troubleshooting.

$ npm run develop
