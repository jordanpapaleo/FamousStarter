DynamicGeometry

DynamicGeometry [constructor] 

DynamicGeometry is a component that defines the data that should be drawn to the webGL canvas. Manages vertex data and attributes.

new DynamicGeometry(options)

Constructor parameters:

NameTypeDescription
optionsObject

instantiation options

Instance methods

.getVertexBuffer(bufferName)

Gets the buffer object based on buffer name. Throws error if bufferName is not provided.

Parameters:

NameTypeDescription
bufferNameString

Name of vertexBuffer to be retrieved.

.setVertexBuffer(bufferName, value, size)

Sets a vertex buffer with given name to input value. Registers a new buffer if one does not exist with given name.

Parameters:

NameTypeDescription
bufferNameString

Name of vertexBuffer to be set.

valueArray

Input data to fill target buffer.

sizeNumber

Vector size of input buffer data.

.fromGeometry(geometry)

Copies and sets all buffers from another geometry instance.

Parameters:

NameTypeDescription
geometryObject

Geometry instance to copy buffers from.

.setVertexPositions(value)

Parameters:

New value for vertex position buffer

.setNormals(value)

.setTextureCoords(value)

Set the texture coordinates on this geometry.
