GeometryHelper

GeometryHelper [constructor] 

A helper object used to calculate buffers for complicated geometries. Tailored for the WebGLRenderer, used by most primitives.

new GeometryHelper()

Instance methods

.generateParametric(detailX, detailY, func)

A function that iterates through vertical and horizontal slices based on input detail, and generates vertices and indices for each subdivision.

Parameters:

NameTypeDescription
detailXNumber

Amount of slices to iterate through.

detailYNumber

Amount of stacks to iterate through.

funcfunction

Function used to generate vertex positions at each point.

.computeNormals(vertices, indices, out)

Calculates normals belonging to each face of a geometry.
Assumes clockwise declaration of vertices.

Parameters:

NameTypeDescription
verticesArray

Vertices of all points on the geometry.

indicesArray

Indices declaring faces of geometry.

outArray

Array to be filled and returned.
