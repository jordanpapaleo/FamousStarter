Instance methods

.start()

Starts the Engine.

.stop()

Stops the Engine.

.isRunning()

Determines whether the Engine is currently running or not.

.step(time)

Updates all registered objects.

Parameters:

NameTypeDescription
timeNumber

high resolution timstamp used for invoking the update method on all registered objects

.loop(time)

Method being called by requestAnimationFrame on every paint. Indirectly recursive by scheduling a future invocation of itself on the next paint.

Parameters:

NameTypeDescription
timeNumber

high resolution timstamp used for invoking the update method on all registered objects

.update(updateable)

Registeres an updateable object which update method should be invoked on every paint, starting on the next paint (assuming the Engine is running).

Parameters:

NameTypeDescription
updateableObject

object to be updated

updateable.updatefunction

update function to be called on the registered object

.noLongerUpdate(updateable)

Deregisters an updateable object previously registered using update to be no longer updated.

Parameters:

NameTypeDescription
updateableObject

updateable object previously registered using update
